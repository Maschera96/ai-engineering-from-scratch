# StyleGAN

> 大多数发生器同时将 `z` 搅拌到每一层中。 StyleGAN 将其分开：首先将 `z` 映射到中间 `w`，然后通过 AdaIN 在每个分辨率级别*注入* `w`。这一单一的改变就解开了潜在的空间，并使真实的面孔成为连续七年解决的问题。

**类型：** 建造
**语言：** Python
**先修：** 阶段 8 · 03 (GANs)、阶段 4 · 08（标准化）、阶段 3 · 07 (CNN)
**时间：** 约45分钟

## 问题

DCGAN 通过一堆转置卷积将 `z` 映射到图像。问题是：`z` 控制一切，姿势、灯光、身份、背景，纠缠在一起。沿`z`的一个轴移动，四个轴都发生变化。您不能要求模型“同一个人，不同的姿势”，因为表示不会以这种方式考虑。

卡拉斯等人。 (2019，NVIDIA) 提议：停止将 `z` 直接输入到卷积层中。输入恒定的 `4×4×512` 张量作为网络输入。学习映射 `z ∈ Z → w ∈ W` 的 8 层 MLP。通过*自适应实例归一化* (AdaIN) 在每个分辨率下注入 `w`：归一化每个卷积特征图，然后通过 `w` 的仿射投影进行缩放和移动。添加每层噪声以获得随机细节（皮肤毛孔、发丝）。

结果：`W` 对于“高级风格”（姿势、身份）与“精细风格”（灯光、颜色）具有大致正交的轴。您可以使用图像 A 的 `w` 表示低分辨率级别，使用图像 B 的 `w` 表示高分辨率级别，从而在两个图像之间交换样式。这解锁了编辑、跨域风格化和整个“StyleGAN-inversion”研究线。

## 概念

![StyleGAN：映射网络+AdaIN+每层噪声](../assets/stylegan.svg)

**映射网络。** `f: Z → W`，8层MLP。 `Z = N(0, I)^512`。 `W` 并不被迫呈高斯分布，它会学习适应数据的形状。

**合成网络。**从学习常数 `4×4×512` 开始。每个分辨率块：`upsample → conv → AdaIN(w_i) → noise → conv → AdaIN(w_i) → noise`。分辨率加倍：4、8、16、32、64、128、256、512、1024。

**阿达因。**

```
AdaIN(x, y) = y_scale · (x - mean(x)) / std(x) + y_bias
```

其中 `y_scale` 和 `y_bias` 来自 `w` 的仿射投影。对每个特征图进行标准化，然后重新设计样式。这里的“风格”是特征图的一阶和二阶统计。

**每层噪声。** 添加到每个特征图的单通道高斯噪声，按学习的每通道因子进行缩放。控制随机细节而不影响全局结构。

**截断技巧。** 在推理时，采样 `z`，计算 `w = mapping(z)`，然后计算 `w' = ŵ + ψ·(w - ŵ)`，其中 `ŵ` 是许多样本的平均值 `w`。 `ψ < 1` 以多样性换取质量。几乎每个 StyleGAN 演示都使用 `ψ ≈ 0.7`。

## StyleGAN 1 → 2 → 3

| 版本 | 年 | 创新 |
|---------|------|------------|
| StyleGAN | 2019年 | 映射网络+AdaIN+噪声+渐进式增长。 |
| StyleGAN2 | 2020年 | 权重解调取代了 AdaIN（修复了液滴伪影）； skip/residual架构；路径长度正则化。 |
| StyleGAN3 | 2021年 | 无别名卷积+等变核；消除了纹理粘附在像素网格上。 |
| StyleGAN-XL | 2022年 | 类条件，1024²，ImageNet。 |
| R3GAN | 2024年 | 更强大的品牌重塑；使用减少 20 倍的参数缩小 FFHQ-1024 上的扩散差距。 |

到 2026 年，StyleGAN3 仍然是以下方面的默认设置：(a) 高 FPS 下的窄域照片写实主义，(b) 少镜头域适应（在具有 100 张图像的新数据集上训练，冻结映射），(c) 基于反转的编辑（找到重建真实照片的 `w`，然后编辑该 `w`）。对于开放域文本到图像，它不是工具，扩散才是。

## 构建它

`code/main.py` 以一维方式实现了一个玩具“style-GAN lite”：一个映射 MLP、一个合成函数，它采用学习的常数向量并使用 `w` 派生的 scale/bias 和每层噪声对其进行调制。它表明通过仿射调制注入 `w` 匹配或击败将 `z` 连接到生成器的输入。

### 第 1 步：绘制网络

```python
def mapping(z, M):
    h = z
    for i in range(num_layers):
        h = leaky_relu(add(matmul(M[f"W{i}"], h), M[f"b{i}"]))
    return h
```

### 步骤 2：自适应实例标准化

```python
def adain(x, w_scale, w_bias):
    mu = mean(x)
    sd = std(x)
    x_norm = [(xi - mu) / (sd + 1e-8) for xi in x]
    return [w_scale * xi + w_bias for xi in x_norm]
```

每个特征图的比例和偏差通过线性投影来自 `w`。

### 步骤3：每层噪声

```python
def add_noise(x, sigma, rng):
    return [xi + sigma * rng.gauss(0, 1) for xi in x]
```

每个通道的 Sigma 是可学习的。

## 常见陷阱

- **液滴伪影。** StyleGAN 1 在特征图中产生了一个斑点液滴，因为 AdaIN 将平均值归零。 StyleGAN 2 的权重解调通过缩放卷积权重来解决这个问题。
- **纹理粘贴。** StyleGAN 1 和 2 纹理遵循像素坐标，而不是对象坐标（插值时可见）。 StyleGAN 3 的无混叠卷积通过加窗 sinc 滤波器解决了这个问题。
- **模式覆盖范围。** 截断 `ψ < 0.7` 看起来很干净，但样本来自窄锥体；如果您需要多样性，请使用 `ψ = 1.0`。
- **反转是有损的。** 将真实照片反转为 `W` 通常是通过优化或编码器（e4e、ReStyle、HyperStyle）来完成。结果在多次迭代中发生漂移。

## 使用它

| 使用案例 | 方法 |
|----------|----------|
| 照片真实的人脸（动漫、产品、狭窄） | StyleGAN3 FFHQ / 定制微调 |
| 从照片中编辑脸部 | e4e 反转 + StyleSpace / InterFaceGAN 方向 |
| 换脸/重演 | StyleGAN + 编码器 + 混合 |
| 阿凡达管道 | StyleGAN3 带 ADA，用于低数据微调 |
| 一些图像的域适应 | 冻结映射网络，微调合成 |
| 多模式或文本条件生成 | 不要，使用扩散 |

对于答案是“人脸照片”的产品级演示，StyleGAN 在推理成本（单次前向传递，4090 上 <10 毫秒）和相同质量条的清晰度方面优于扩散。

## 关键术语

| 学期 | 人们怎么说 | 它实际上意味着什么 |
|------|-----------------|-----------------------|
| 测绘网络 | “MLP” | `f: Z → W`，8 层，将潜在几何与数据统计解耦。 |
| W空间 | 「风格空间」 | 映射网络的输出；大致解开。 |
| 阿达因 | “自适应实例规范” | 标准化特征图，然后通过 `w` 投影进行缩放 + 移动。 |
| 截断技巧 | “PSI” | `w = mean + ψ·(w - mean)`，ψ<1 以多样性换取质量。 |
| 路径长度正则化 | “PL 规则” | 对 `w` 中每单位变化图像的较大变化进行惩罚；使`W`更流畅。 |
| 重量解调 | “StyleGAN2 修复” | 标准化转换权重而不是激活；消除液滴伪影。 |
| 无别名 | 《StyleGAN3的伎俩》 | 带窗 sinc 滤波器；消除了纹理粘附在像素网格上。 |
| 反转 | “寻找真实图像的 w” | 优化或编码 `x → w`，因此 `G(w) ≈ x`。 |

## 生产说明：为什么 StyleGAN 仍会在 2026 年发货

4090 上的 StyleGAN3 在 10 毫秒内生成 1024² FFHQ 人脸 — `num_steps = 1`，无 VAE 解码，无交叉注意力传递。在生产方面，这是任何图像生成器的底层延迟。相同分辨率下的 50 步 SDXL + VAE 解码管道大约需要 3 秒。这是一个 **300× 的差距**，对于窄域产品（化身服务、ID 文档管道、股票面孔生成），它在 TCO 方面获胜。

两个操作后果：

- **没有调度程序，没有批处理程序。** 目标占用率的静态批处理是最佳的。连续批处理（对于 LLM 和扩散至关重要）提供的收益为零，因为每个请求都采用相同的 FLOP。
- **截断 `ψ` 是安全旋钮。** `ψ < 0.7` 从映射网络范围的窄锥体中采样。这是服务层具有超过样本方差的唯一杠杆。在峰值负载时降低 `ψ`，为高级用户提高它。
