# 投票、自洽和辩论拓扑
> 最便宜的聚合：采样 N 个独立代理，多数投票。王等人。 2022 年的自我一致性通过对一个模型进行 N 次采样来做到这一点。多智能体通过**异质**智能体对其进行扩展，以摆脱单一文化——不同的模型、不同的提示、不同的温度、不同的环境。除了多数投票之外，辩论拓扑也很重要：MultiAgentBench（arXiv：2503.01935，ACL 2025）评估了星/链/树/图协调，并发现**图最适合研究**，对过去的〜4个代理征收“协调税”。 AgentVerse (ICLR 2024) 记录了两种新兴模式——志愿者行为和从众行为——而从众行为既是一种特征（寻求共识）也是一种风险（群体思维，第 24 课）。本课程绘制拓扑空间、构建每个变体并测量协调税。
**类型：** 学习 + 构建
**语言：** Python (stdlib)
**先决条件：** 第 16 阶段·07（思想与辩论社会），第 16·14 阶段（共识和 BFT）
**时间：** ~75 分钟
＃＃ 问题
辩论可以提高准确性（Du et al., arXiv:2305.14325）。它还可以降低它的性能。辩论是否有帮助取决于四个结构性选择：
1. 谁与谁通话（拓扑）。
2. 多少轮（Du 2023：轮次和代理都是独立的）。
3.智能体是否异质（不同的基础模型打破单一文化）。
4. 是否存在对抗性声音（钢人与稻草人）。
与单个代理相比，“运行 5 个代理并投票”执行任务的团队通常会出现退化。失败不是随机的。他们跟踪拓扑和异构性。本课是拓扑图。
＃＃ 概念
### 自我一致性，单一模型基线
王等人。 2022（“自我一致性改进了思维推理链”）在温度 > 0 的情况下对同一模型进行了 N 次采样，并对推理路径答案进行了多数投票。 GSM8K 上的结果：在单次贪婪解码中，N=40 个样本获得了显着的增益。自我一致性是单智能体投票的前身。
限制：自洽使用一个基本模型。错误与构造相关。如果模型存在系统偏差，则所有 N 个样本都会共享该偏差。
### 多智能体投票，异构扩展
用 N 个*不同的*试剂替换 N 个样本。不同的基础模型（Claude、GPT、Llama）、不同的提示、不同的工具访问。好处：不相关的错误。费用：不同的代理费用不同；协调它们会增加开销。
2026 年异构辩论的规范名称是 **A-HMAD** — 对抗性异构多主体辩论。虽然没有被普遍采用，但论文使用这个术语来表示“不同模型的辩论，这减少了单一文化崩溃带来的相关错误”。
### 四种拓扑
```
star                chain               tree                graph

    ┌─A─┐           A─B─C─D         ┌──A──┐              A───B
    │   │                           │     │              │ × │
    B   C                           B     C              D───C
    │   │                          / \   / \
    D   E                         D   E F   G           (fully connected)
```

星号：一个集线器，所有其他集线器仅与集线器通信。相当于没有后台通道的supervisor-worker。
Chain：线性，每个智能体都能看到前一个智能体的输出。类似管道。
树：分层的，由分层代理系统使用（第 06 课）。
图：任意对任意。包括完全连接的派系和任意 DAG。
### 协调税 (MultiAgentBench)
MultiAgentBench（MARBLE、ACL 2025、arXiv：2503.01935）对任务套件上的星形、链形、树形、图进行基准测试，包括研究、编码和规划。主要测量结果：
- **图**拓扑在研究任务中获胜。信息在任何地方流动；代理可以互相批评。
- **明星**在快速回答事实性任务中获胜。集线器过滤并合并。
- **链**在逐步管道上获胜（分阶段细化）。
- **协调税**出现在图拓扑中超过约 4 个代理。挂钟和代币成本的增长速度快于质量的增长速度。
4 名代理人的上限是经验性的，而不是根本性的。它反映了 2026 年 LLM 上下文容量：每个代理的上下文都充满了同行的输出，一旦每个人都可以看到每个人，添加代理 N+1 的边际价值就会下降。
### 多代理辩论策略（“我们应该发疯吗？”）
arXiv:2311.17371 是 2023 年 MAD 策略调查。其他人重复的关键发现：在使用相同预算时，与自我一致性“结构相似”（独立采样+聚合）的 MAD 变体通常表现不佳。当代理人确实具有异质性并且辩论具有对抗性结构（一名代理人反对）时，MAD 的帮助最大。
### AgentVerse 涌现模式
AgentVerse（ICLR 2024，https://proceedings.iclr.cc/paper_files/paper/2024/file/578e65cdee35d00c708d4c64bce32971-Paper-Conference.pdf）记录了多智能体辩论中出现的两种行为，即使没有明确的设计：
- **志愿者。** 代理会主动提供帮助（“我可以采取下一步”）。有用：它将工作分配给最有能力执行子任务的代理。
- **从众。** 代理人会调整自己的立场以适应批评者，即使批评者是错误的。这相当于辩论中的阿谀奉承（第 14 课）。
服从就是为什么辩论直至达成协议会奖励恶霸。由单独裁判进行的有界回合会减轻影响。
### 异质性：移动精度的实际旋钮
实用文献中的 2024-2026 年模式：将 N 个代理之一替换为不同的基本模型，比将 N 增加 1 能带来更大的准确度提升。直觉是单一文化——每个新的独立误差源都比额外的相关样本更有价值。
在极限情况下，异质性战胜了数量。在大多数具有明确基础事实的任务中，三种不同的模型击败了一种模型的五个副本。
### 评审方法
Sibyl 框架（在 Minsky-LLM 文献中引用）正式形成了一个“陪审团”——一小群专门的代理，通过在每个阶段的投票来完善答案。与简单的多数投票不同，陪审团有自己的角色：一名代理人进行盘问，一名代理人提供背景，一名代理人对合理性进行评分。陪审团方法是介于普通投票（廉价、倾向于单一文化）和完全 MAD（昂贵、倾向于从众）之间的中间点。
### 当投票与辩论占主导地位时
- 问题具有基本事实（事实、数学、代码行为）。投票趋同是有意义的。
- 代理可以访问不同的来源或工具（可以使用异构性）。
- 回合有限制（通常为 2-3 轮），并且有单独的裁判或验证员。
- 预算允许 3-5 名代理。在图拓扑上超过 5-7 时，协调税占主导地位。
### 当投票与辩论带来伤害时
- 这个问题是意见型的。特工们会倾向于选择看起来最有信心的答案，而不是最正确的答案。
- 所有代理共享一个基本模型。单一文化使共识变得毫无意义。
- 回合数不受限制。一致性每次都会获胜。
- 任务很简单。在 N=5 时具有自我一致性的单个代理更便宜且更准确。
## 构建它
`code/main.py` 实现：
- `run_star(agents, hub, question)` — 集线器轮询每个工作人员，进行聚合。
- `run_chain(agents, question)` — 顺序细化。
- `run_tree(root, children, question)` — 具有深度 2 聚合的分层结构。
- `run_graph(agents, question, rounds)` — 全面辩论，有界回合。
- 脚本化的异质性表盘：每个代理都有一个 `error_bias` 指示其系统错误。
- 测量工具，以 N=3、5、7 运行每个拓扑并报告（准确性、total_tokens、wallclock_simulated）。
跑步：
```
python3 code/main.py
```

预期输出：拓扑×N→（准确度、标记、延迟）的表。 Graph 在研究型任务中以 N=3-5 获胜；明星在快速事实性任务中获胜； N=7 处的图表显示了协调税（延迟比准确性增长得更快）。
## 使用它
`outputs/skill-topology-picker.md` 是一种读取任务描述并推荐拓扑（星型/链型/树型/图）、N（代理数量）、异质性配置文件（要使用的基本模型）和轮次边界的技能。
## 发货
对于任何合奏：
- 使用一个强大的基础模型从 **N=5** 的自我一致性开始。这是便宜的基线。
- 如果准确性很重要，则升级到 **N=3** 的异构投票。测量增量。
- 仅当任务具有结构（研究、多步骤）并且有界回合可行时，才升级到**辩论拓扑**。
- 始终记录少数集群。当少数人坚持正确时，就会发出多元化信号。
- 基准挂钟和令牌以及准确性。 “以 10 倍的成本获得更高的准确性”是一个商业决策。
## 练习
1.运行`code/main.py`。绘制图拓扑的协调税曲线：准确度与 N、令牌与 N。曲线在 N 时发生拐点？
2. 实施 A-HMAD：三个具有故意不同偏见的代理人。在第 14 课的单一文化攻击中，全相同偏差基线与 A-HMAD 相比如何？
3. 在图拓扑中添加一个“法官”角色，该角色不投票，只对最终共识进行评分。这会改变紧急的从众行为吗？
4. 阅读 AgentVerse 论文 (ICLR 2024)。确定您的实施表现出最强烈的紧急行为。你能通过迅速改变来引发相反的行为吗？
5. 阅读 MultiAgentBench (arXiv:2503.01935) 第 4 节（拓扑实验）。使用您的工具从论文中重现一项任务的“图胜研究”结果。
## 关键术语
|术语 |人们怎么说|它实际上意味着什么 ||------|----------------|------------------------|
|自我一致性| “样品N次，投票”| Wang 2022。单一模型，N 个温度>0 样本，多数对推理路径进行投票。 |
|异质性| “不同型号”|不同基本模型或提示系列的合奏。打破单一文化。 |
|疯狂 | “多智能体辩论”|代理人在轮次中交换批评意见的通用术语。参见杜 2023。
| A-HMAD | “对抗性异构 MAD”| MAD 变体强调不同模型+对抗结构。 |
|拓扑 | “谁跟谁说话”|星、链、树、图。决定信息流。 |
|协调税| “收益递减”|图中~4 个代理以上，成本增长速度快于质量增长速度。 |
|志愿者行为| “无提示的帮助”| AgentVerse 紧急模式：代理提出采取步骤。 |
|合规行为 | “压力下的协议”| AgentVerse 涌现模式：代理人与批评者结盟。 |
|评审团| “小型专用面板”|具有角色的西比尔风格合奏（考官、背景、记分员）。 |
